    Visual Recognition

    IBM waston 视觉认知服务基于深度学习算法来分析场景，图片，目标，人脸和其他内容。你还可以训练你自己的图片集，然后上传图片来寻找相似的图片。

    服务流程

    VR服务是基于一系列内建函数的，因此你可以调用服务来达到很高的图片识别精度。你也可以训练你自己的类别，来寻找相似的图片。
    [如图]（http://www.ibm.com/watson/developercloud/doc/visual-recognition/images/vr-process.png）

    关于图片分类的教程

    这个教程会指导你如何使用默认的分类器来识别图片以及人脸

    开始之前确保你有一个bluemix账号

    第一步 得到服务凭证

    · 1.登陆bluemix
    · 2.创建一个服务实例，
    在bluemix面板，选择服务，在添加服务下，服务名那一栏为你的服务起一个名字，其他选项不要管。然后点击创建按钮

    · 3.复制你的凭证
    点击服务凭证来查看你的凭证，并且把它复制。

    第二步 分类一张图片

    VR服务使用一个强大的内建函数使得你可以快速低耗的分析一张图片。你可以提交一个网址或是图片链接，或是直接上传一张图片（最大不能超过2M）。 ·1.用POST
    /v3/classify 方法提交的图片会被识别分析，下面的例子是是上传了一张图片并且用内建函数进行分析的

    API会返回一个响应，内容包括默认分类器，图片中的类别，以及每一个类别的得分。得分从0到1，越高的分数说明越准确。默认分类器省略了低分类别，不过你可以添加进来，只要在设置show_low_confidence参数为1即可。

    人脸识别和这个大同小异，不过你要调去的是POST /v3/detect_faces这个方法。

    创建应用实例获得认证信息。
    在认证信息的页面左侧，选择服务认证，获取API key，
    用你的API key替换{api-key}

    方法

    分类图片

    默认情况下，上传图片或者是URL连接到定义的分类器。
    如果用自定义的分类器，那么必须包括classifier_ids和owners参数，图片必须是jpeg或者是png格式的。
      对于每一张图片来说，响应包括选择分类器对于每一类的得分，分数在0-1之间得分越高表明图片中的相似度越大。返回结果的分数至少是0.5.

    参数及说明
    api_key 你的API key

    version 你想要使用的API版本，格式是YYYY-MM-DD，当前最新的版本是2016-05-20

    images_file 要分类的图片（jpg或是png）或是压缩包.压缩包的上限是20张，5M

    parameters JSON格式的输入参数，包括URL：待分析图片的网址[字符串]，classifier_ids:选用的分类器[数组]，owners: 分类器的拥有者，IBM或是me[数组]
    阈值：显示返回分类样本结果的最小值[浮点数]

    Accept-Language 输出的语言，英语（en），西班牙语（es），阿拉伯语（ar），日语（ja），如果没有选择，就会忽略该参数

    响应的格式

    images 分类的图像数组

    classifiers 分类器数组

    class 图像中识别的类

    score 每一类的评分

    classifier_id 图像分类使用的分类器

    name 识别出的类别名

    image 相对路径，如果通过URL就省略

    source_url 重定向之前的图片URL网址，上传的会shengl

    resolved_url 重定向之后的，完全路径

    images_processed 处理的图片数量

    error 错误的信息

    error_id 错误名字

    description 错误的描述

    warnings 警告 引起输出不是最优的原因
