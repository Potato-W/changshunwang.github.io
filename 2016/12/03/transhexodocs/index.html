    翻译hexo官方的教程

    安装

    Hexo安装后，运行以下命令在新的文件夹中创建你的Hexo。
    ‘’’ bash
    $ hexo init <文件夹名>
    $ cd <文件夹名>
    $ npm install
    ‘’’ bash

    创建后，文件夹的目录应该是这样的：
    |—— _config.yml
    |—— package.json
    |—— scaffolds
    |—— source
    | |—— _drafts
    | | _posts
    |     themes

    接下来详细看下各个文件夹

    _config.yml

    你可以在这里做一些关于网站基本的配置

    Site

    title 网页的名称
    subtitle 网页的副标题
    description 网页的简介
    author 作者名称
    language 网站使用的语言，默认是英语en，中文是zh，日文是ja
    timezone 网页的时区设置，Hexo会默认设置你计算机设置的时间。支持的时区可以在
    这里找到，

    URL

    url 你的网址
    root 网址的根目录
    permalink 文章的固定连接格式，更多看这里
    permalink_default 固定连接中每一段的初始默认值
